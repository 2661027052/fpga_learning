# 竞争与冒险

## 问题

> 什么是竞争？什么是冒险？有哪些类型？如何消除冒险？

> 难度: ★ | 频率: 高频 | 层级: L1

## 结论

竞争 = 信号到达时间差；冒险 = 输出毛刺。**有竞争不一定有冒险，有冒险一定有竞争。**

## 分类

| 类型 | 描述 | 典型电路 |
|------|------|---------|
| 静态1冒险 | 应1变0毛刺 | 与门两输入不同时翻转 |
| 静态0冒险 | 应0变1毛刺 | 或门两输入不同时翻转 |
| 动态冒险 | 多次跳变 | 多级组合延迟差大 |

## 4种修复（按推荐度）

1. **DFF打拍** ← 最推荐，毛刺窄于时钟周期
2. 加冗余项 ← 卡诺图相邻1同覆盖
3. 选通脉冲 ← 毛刺过后再打开
4. 滤波电容 ← PCB可用，FPGA不可

## 面试必答

"FPGA中最推荐的方法是在组合逻辑输出后加一级 D 触发器——毛刺宽度通常 <1ns，触发器只在时钟沿采样，天然滤除。这也是为什么三段式状态机第三段用时序输出的原因。"

## 补充：仿真看不到毛刺

前仿真用理想延迟模型，看不到实际门延迟产生的毛刺。必须跑后仿真（Post-Synthesis / Post-PnR）才能暴露。
