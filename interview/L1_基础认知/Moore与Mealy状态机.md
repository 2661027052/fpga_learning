# Moore 与 Mealy 状态机

## 问题

> Moore 和 Mealy 状态机的区别是什么？各有什么优缺点？FPGA 中推荐哪种？

> 难度: ★ | 频率: 高频 | 层级: L1

## 结论

Moore 输出只与当前状态有关；Mealy 输出与当前状态+输入有关。Moore 输出稳定但状态多，Mealy 输出快但可能有毛刺。

## 对比

| | Moore | Mealy |
|------|-------|-------|
| 输出决定 | f(当前状态) | f(当前状态, 输入) |
| 输出时序 | 同步于时钟沿 | 输入变化立即反映 |
| 毛刺风险 | 无(时序输出) | 有(组合输出) |
| 状态数 | 多 | 少 |
| 响应速度 | 慢一拍 | 即时 |
| FPGA推荐 | 控制路径 | 数据路径 |

## 选择指南

- 要求输出无毛刺 → Moore + 三段式时序输出
- 要求快速响应 → Mealy + 输出寄存器打拍消毛刺
- FPGA 默认推荐 Moore 三段式
