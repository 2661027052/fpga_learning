# FPGA 面试题库

与本仓库 L1-L4 代码模块同步的面试问答，按难度层级组织。

## 结构
```
interview/
├── L1_基础认知/    ★★         FPGA概念、数字电路基础
├── L2_核心技能/    ★★~★★★    Verilog、状态机、时序基础、CDC基础
├── L3_进阶应用/    ★★★~★★★★  总线协议、异步FIFO、时序约束、复位
└── L4_专家领域/    ★★★★~★★★★★ DDR、低功耗、SystemVerilog、高速接口
```

## 答题模板
