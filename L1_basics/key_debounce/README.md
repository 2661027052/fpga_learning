# 按键消抖模块 (key_debounce)

## 知识点
Verilog 基础语法 - 边沿检测 + 按键消抖

## 功能描述
- 处理异步按键输入，消除机械抖动
- 采用两级同步器消除亚稳态
- 20ms 计数器消抖，确保按键信号稳定后才确认有效
- 边沿检测产生单周期高电平脉冲

## 模块端口

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| clk | 输入 | 1 | 系统时钟 50MHz |
| rst_n | 输入 | 1 | 异步复位，低电平有效 |
| key_in | 输入 | 1 | 异步按键输入（低电平表示按下） |
| key_pressed | 输出 | 1 | 按键按下脉冲（单周期高电平） |

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| DEBOUNCE_CNT | 1_000_000 | 消抖计数器阈值，对应 20ms @ 50MHz |

## 模块架构

```
key_in → [两级同步器] → [消抖计数器] → [边沿检测] → key_pressed
```

## 仿真说明

```bash
cd sim
iverilog -o tb_key_debounce.vvp tb_key_debounce.v ../rtl/key_debounce.v
vvp tb_key_debounce.vvp
gtkwave tb_key_debounce.vcd
```

测试平台将 `DEBOUNCE_CNT` 覆写为 20，模拟 4 种场景：
1. **按键按下含抖动** - 快速翻转后稳定低电平，验证产生脉冲
2. **按键释放含抖动** - 验证不产生错误脉冲
3. **第二次按下** - 验证模块可重复使用
4. **短毛刺干扰** - 验证短脉冲被正确滤除

## 设计要点
- 两级同步器：`key_sync0` → `key_sync1`，降低亚稳态概率
- 消抖计数器在信号与稳定值不同时开始计数，计数达到阈值才确认变化
- 若信号在计数过程中恢复原值，计数器立即清零（毛刺滤除）
- 下降沿检测：`key_falling = ~key_stable & key_stable_d1`
